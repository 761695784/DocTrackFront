Ajouter un utilisateur

Le rôle est requis.
Le nom est requis. Le nom ne peut pas dépasser 40 caractères.
Le prénom est requis. Le prénom ne peut pas dépasser 20 caractères.
L'email est requis. L'email doit être valide. L'email ne peut pas dépasser 50 caractères.
Le mot de passe est requis. Le mot de passe doit contenir au moins 8 caractères.
Les mots de passe ne correspondent pas.
Le numéro de téléphone est requis. Le numéro de téléphone ne peut pas dépasser 20 caractères.
L'adresse est requise. L'adresse ne peut pas dépasser 100 caractères.
